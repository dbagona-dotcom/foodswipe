FoodSwipe 🔥
Mobilní aplikace fungující jako Tinder pro objevování restaurací v reálném čase.

Technologie:
Dart & Flutter
Google Places API (REST API komunikace, parsování JSON)
Geolocator (získávání GPS souřadnic uživatele)
SharedPreferences (lokální ukládání dat na zařízení)

Hlavní funkce:
Dynamické načítání reálných dat o podnicích v okolí pomocí Google API.
Animované swipování karet (💚 líbí / ❌ nelíbí).
Nastavení radiusu vyhledávání (1–20 km) s využitím zpožďovače (debounce) proti spamu.
Vlastní algoritmus pro filtrování kategorií na základě mapování klíčových slov.
Ukládání oblíbených restaurací a jejich správa v paměti zařízení.

FoodSwipe 🔥
A mobile app that works like Tinder for real-time restaurant discovery.

Technology:
Dart & Flutter
Google Places API (REST API communication, JSON parsing)
Geolocator (obtaining user's GPS coordinates)
SharedPreferences (local storage of data on the device)

Main features:
Dynamic loading of real-time data about nearby businesses using Google API.
Animated card swiping (💚 like / ❌ dislike).
Setting the search radius (1-20 km) using a debouncer against spam.
Custom algorithm for filtering categories based on keyword mapping.
Saving favorite restaurants and managing them in the device's memory.
